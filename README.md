# SokobanPlus
A clone of the Japanese classic Sokoban, with an undo/redo feature. Created for a Game Engineering class to familiarize myself with the [Command](https://gameprogrammingpatterns.com/command.html) pattern.

# How to Play
## Controls
**WASD** to move up, down, left, right

**Z** to undo

**X** to redo

**R** to restart the level

**ESC** to quit
## Objective
Collect the gem by moving your character throughout the level. You can push boxes out of your way, but the solid squares are walls and cannot be moved.
